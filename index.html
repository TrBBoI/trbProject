Echoknights

Project Description

DigiPen Junior Year Game Project. 3D MOBA running on OpenGL.

Echo Knights is an arena-style shoot 'em up game where up to 4 players compete against each other to be the last one standing. Echoes of player's past will either assist or hinder them at the start of each game. It is up to players to plan ahead and make full use of their past to influence their future.

  • An awesome 3D Battle Arena Game running on OpenGL and glsl.

What I did

Role: Graphics Programmer
  • Sci-fi graphics enhanced with multi-pass bloom
  • Implemented dynamic post process rendering system makes adding post process render easy
  • Implemented task-oriented rendering, a simple yet efficient 3D rendering
  • Added object silhouetting so objects behind wall will not be unseen
  • Introduced auto-reload shader program so you can change the shader on the fly
Role: Tools Programmer
  • Implemented a debug console to support game debugging
  • Implemented a debug logger to ease debugging process
Role: Technical Director
  • Directed a 8 members team

Dreamwalker

Project Description

DigiPen Sophomore Year Game Project. 2D Puzzle-based Game running on Directx9.

DreamWalker is a puzzle platformer set in a dream-like world where you get to experience infinite possibilities within a finite space. You play as Warance, a guy who has no recollection of what happened to him before he came into this dream world. Your main goal is to collect all his memories and solve puzzles to find out what had happened to him!

DerrFex

Project Description
